# 🛎️ Coupang Customer Support Management Center

쿠팡 고객상담관리센터는
고객센터 상담원이 고객 정보와 상담 이력을 조회하고 관리하기 위해 사용하는 **내부 관리자용 웹 시스템**입니다.

상담 현황을 대시보드 형태로 제공하여 상담 진행 상황을 한눈에 파악하고 효율적으로 관리할 수 있도록 설계했습니다.

<br/>

# 🖥️ 실행 환경

* 🌐 최신 웹 브라우저 (Chrome 권장)
* ⚙️ 별도 서버 및 추가 설치 없이 실행 가능

<br/>

# 🚀 실행 방법

1️⃣ 프로젝트 폴더에서 **index.html** 파일을 실행합니다.  
2️⃣ 브라우저에 **고객상담관리센터 화면이 표시되면 정상 실행된 것입니다.**

<br/>

# ✨ 주요 기능

## 📊 상담 현황 대시보드

오늘 상담 데이터를 카드 UI 형태로 표시합니다.

* 📥 오늘 접수 상담 수 확인
* ⏳ 처리 대기 상담 확인
* 🔄 처리 중 상담 확인
* ✅ 상담 처리 완료 건수 확인

<br/>

## 📋 상담 관리 기능

* 📂 상담 문의 목록 조회
* 🔎 상담 상세 정보 확인
* ✏️ 상담 상태 관리

<br/>

## 📢 공지사항 안내

시스템 점검이나 중요한 안내사항을 공지 영역에서 확인할 수 있습니다.

예시

> 📢 시스템 점검 예정
> 오늘 23:00 ~ 23:30 동안 접속이 제한될 수 있습니다.

<br/>

# 🔄 기본 사용 흐름

1️⃣ **대시보드에서 상담 현황을 확인합니다.**  
2️⃣ **문의 목록에서 상담 내역을 조회합니다.**  
3️⃣ **상담 상세 페이지에서 상담 내용을 확인합니다.**  
4️⃣ **상담 상태를 변경하여 처리 진행 상황을 관리합니다.**  

<br/>

# ⚠️ 이상 상황 안내

화면이 정상적으로 표시되지 않을 경우 다음 사항을 확인합니다.

* 🔄 브라우저 새로고침
* 🌐 최신 브라우저 사용 여부 확인
* 📂 파일 경로 확인

<br/>

# ❓ FAQ

### Q. 설치가 필요한가요?

A. 별도 설치 없이 **웹 브라우저에서 바로 실행**할 수 있습니다.



### Q. 어떤 브라우저에서 실행할 수 있나요?

A. Chrome, Edge 등 **최신 웹 브라우저에서 실행**할 수 있습니다.
